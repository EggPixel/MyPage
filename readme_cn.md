## Contribution  
首先，进入 "jar/Version.json" 编辑它,使用 ","来分割各个版本.  
然后, 进入 "jar/\<Server Type\>/\<Server Version\>/" 新建文件 index.json.  
文件格式:  
{  
    "Hash": "\<Hash\>",  
    "Path": "\<Jar Path\>",  
    "Type": "\<Type\>",  
    "Version": "\<Version\>"  
}  
最后 新建pr提交它 然后等我来确认
